## What Was Built

Implemented backend archive filtering to ensure archived notes are filtered at the database level (not just frontend):

### Backend Changes (NoteRepository.java)
- Added new method `findByUserIdAndIsArchivedFalse(UUID userId, Pageable pageable)`
- This returns only non-archived notes for a user

### Backend Service Changes (NoteServiceImpl.java)
- Updated `getAllNotes()` to use `findByUserIdAndIsArchivedFalse()` instead of `findByUserId()`
- Main note list now filters archived notes at the database level
