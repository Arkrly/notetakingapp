# Plan 02: JWT Validation Tests

## Objective

**Phase:** 03-security-authentication-audit  
**Goal:** Verify JWT tokens are properly validated  
**Purpose:** Confirm "none" algorithm and expired tokens are rejected  

## Context

@.planning/phases/03-security-authentication-audit/03-CONTEXT.md

### Current State

- JwtUtils uses `Jwts.parser().verifyWith(signingKey).build().parseSignedClaims(token)`
- This requires a signature - unsigned tokens should fail
- Expiration is checked via `isTokenExpired()`
- **Gap:** No explicit tests covering security scenarios

### Expected Behavior

- SEC-03: JWT tokens with "none" algorithm should be rejected
- SEC-04: JWT tokens with expired timestamp should be rejected

### task 2: Add unit tests for JWT security scenarios

<task type="auto">
<parameter name>task 2: Add JWT security unit tests</name>
<files>notetakingapp/src/test/java/com/notetakingapp/security/JwtUtilsTest.java</files>
<action>
Create JwtUtilsTest.java with the following test cases:

1. **testRejectNoneAlgorithm**: 
   - Create JWT with "none" algorithm using Jwts.builder().signWith(Jwts.SIG.NONE)
   - Call isTokenValid()
   - Assert returns false

2. **testRejectExpiredToken**:
   - Create JWT with expired timestamp (exp = past date)
   - Call isTokenValid()
   - Assert returns false

3. **testAcceptValidToken** (sanity check):
   - Create valid JWT with proper signature
   - Call isTokenValid()
   - Assert returns true

Use jjwt 0.12.x API:
```java
// For none algorithm test
String noneToken = Jwts.builder()
    .subject("testuser")
    .expiration(new Date(System.currentTimeMillis() + 3600000))
    .signWith(Jwts.SIG.NONE)
    .compact();

// For expired token
String expiredToken = Jwts.builder()
    .subject("testuser")
    .expiration(new Date(System.currentTimeMillis() - 1000)) // expired
    .signWith(signingKey)
    .compact();
```
</action>
